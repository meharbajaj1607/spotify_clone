# Spotify Web UI Clone

A static clone of the Spotify Web Player interface built using HTML and CSS.  
This project focuses on layout accuracy, UI structure, and styling consistency inspired by the original Spotify design.

## Features
- Spotify-like navigation bar and sidebar
- Responsive flexbox-based layout
- Custom fonts and icons
- Clean and structured UI components

## Tech Stack
- HTML5
- CSS3
- Flexbox

## Project Status
This project is actively being improved.  
Planned enhancements include JavaScript-based interactivity and responsive behavior for smaller screens.

## How to Run
1. Clone the repository
2. Open `index.html` in any modern web browser
3. Save picturess in folder named `assests`

- Improve mobile responsiveness
